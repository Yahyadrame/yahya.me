## Développement

Clonez le dépôt, installez les dépendances et lancez l'application

```bash
git clone git@github.com:Yahyadrame/yahya.me.git
cd yahya.me
npm install
npm start

# Pour le build et le service de pages statiques
npm run build
npm -g install serve
serve build
```

**Note :** utilisez le paramètre `depth` pour réduire la taille du clone et accélérer l'opération.

```sh
git clone --depth=1 https://github.com/Yahyadrame/yahya.me.git
```
